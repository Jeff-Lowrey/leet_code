How to Add a Solution

1. Choose Your Language

Solutions can be written in any of these supported languages:

  • Python
  • JavaScript
  • Java
  • C++
  • C
  • TypeScript
  • Go
  • Rust
  • C#
  • Swift

2. Create Your Solution File

Place your solution in the appropriate directory:

solutions/{{ category }}/<language>/{{ filename }}.<ext>

Example: solutions/{{ category }}/python/{{ filename }}.py

3. Follow the Template

Use the standard solution template for your language:

  • Python Template
  • JavaScript Template

Templates include sections for:

  • Problem description
  • Approach explanation
  • Time/space complexity
  • Solution implementation
  • Test cases

4. Documentation Reference

For detailed guidelines, see:

  • Contributing Guide
  • Solution Format Guide
